# 🌤 SkyCast – Weather Intelligence

SkyCast is a modern and responsive weather web app that provides real-time weather data, location-based results, and smart city suggestions.

---

## 🚀 Features

- 🌍 Real-time weather data
- 📍 Get weather using your current location
- 🔍 Smart city search with suggestions
- 🌡 Displays temperature, humidity, wind, visibility
- 🎨 Clean glassmorphism UI
- 📱 Fully responsive design

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript
- WeatherAPI
